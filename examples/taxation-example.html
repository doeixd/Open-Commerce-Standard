OCP Taxation Example

This example demonstrates client-side tax estimation using the dev.ocp.store.taxation@1.1 capability, combined with product-specific tax codes from dev.ocp.product.taxation@1.0.

Client Implementation

The client discovers taxation capabilities and calculates taxes locally:

// 1. Discover store taxation capability const capabilities = await fetch('/capabilities'); const taxationCap = capabilities.capabilities.find( c => c.id === 'dev.ocp.store.taxation@1.1' ); // 2. Get taxation policy const taxationPolicy = await fetch(taxationCap.schemaUrl); // 3. For each product, check tax code const products = await fetch('/catalogs/default'); products.items.forEach(item => { const taxCode = item.metadata?.['dev.ocp.product.taxation@1.0']?.taxCode; // Use taxCode to look up rate from taxationPolicy.productTaxCodes }); // 4. Calculate taxes client-side function calculateTax(subtotal, shipping, regionCode, taxCodes) { const region = taxationPolicy.supportedRegions.find(r => r.regionCode === regionCode); const baseTaxRate = region.rate / 100; let totalTax = 0; // Apply product-specific tax rates taxCodes.forEach(({code, amount}) => { const taxCode = taxationPolicy.productTaxCodes.find(tc => tc.taxCode === code); const rate = taxCode ? taxCode.rate / 100 : baseTaxRate; totalTax += amount * rate; }); // Add shipping tax if applicable if (region.taxOnShipping) { totalTax += shipping * baseTaxRate; } return totalTax; }

Key Benefits

  • Client-Side Estimation: Users see final prices immediately without server round-trips
  • Product-Specific Rates: Different tax treatment for food, luxury goods, etc.
  • Regional Compliance: Automatic handling of different tax jurisdictions
  • Real-Time Updates: Tax calculations update as users change shipping addresses
  • Server Authority: Final tax calculation still performed server-side for accuracy
